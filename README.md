# Laravel_EC
基本的な機能を実装したECサイトです

## 動作環境
PHP 7.3  
Laravel 5.8  
DB: ローカル MariaDB 10.4.6 デプロイ版 MySQL 5.7 Community

## デプロイ版
<http://provisoriam.site/>  

**注意点**  
* 日本国内からのみアクセス可能です  
* メールサーバーを設定してないのでパスワード再設定は利用できません  
* バグが多数存在すると思われます  
* 一応パスワードはハッシュ化、住所は暗号化してますが、普段使用してるパスワードや本当の住所等を入力しないでください
